# Context for gwt-0009: fork.server.no-regression-normal-commands

## Behavior
Given `mapCliOptionsToSDK` receives a normal (non-fork) options object that does NOT include `resume`, `forkSession`, `resumeSessionAt`, or `sessionId`, when it constructs `sdkOptions`, then `sdkOptions` is identical to what it produced before this change — `cwd`, `permissionMode`, `allowedTools`, `disallowedTools`, `model`, `systemPrompt`, `tools`, `settingSources` are set as before, and no fork fields appear.

## Concrete Data Shapes
```ts
type NormalCliOptions = {
  cwd?: string;
  permissionMode?: 'default' | 'plan' | 'bypassPermissions' | 'acceptEdits';
  toolsSettings?: { allowedTools: string[]; disallowedTools: string[]; skipPermissions: boolean };
  model?: string;                // default 'sonnet'
  appendSystemPrompt?: string;
  // No resume, forkSession, resumeSessionAt, sessionId.
};

// Expected sdkOptions keys (exhaustive for normal path):
//   cwd?, permissionMode?, allowedTools, disallowedTools, tools, model,
//   systemPrompt, settingSources.
// MUST NOT contain: resume, forkSession, resumeSessionAt, sessionId.
```

## Key Invariants to Model
- NoForkFieldsWhenAbsent: none of `resume`/`forkSession`/`resumeSessionAt`/`sessionId` appear as own-properties in `sdkOptions` when absent from input.
- NormalShapePreserved: output key-set is a subset of the pre-change legacy key-set for normal input.
- PlanModeToolsUnion: when `permissionMode === 'plan'`, `allowedTools` ⊇ `{Read, Task, exit_plan_mode, TodoRead, TodoWrite, WebFetch, WebSearch}`.

## Modeling Approach
- CONSTANTS: PLAN_TOOLS = {Read, Task, exit_plan_mode, TodoRead, TodoWrite, WebFetch, WebSearch}.
- Variables: `input` (normal-shape record), `output`.
- Actions: `Map(input)` → compute output.
- Invariants: NoForkFieldsWhenAbsent, NormalShapePreserved, PlanModeToolsUnion.

## Test Interface (MANDATORY — concrete code)
```ts
// cc-agent-ui/tests/generated/gwt-0009.test.ts
import { describe, it, expect } from 'vitest';
import { mapCliOptionsToSDK } from '../../server/claude-sdk.js';

describe('gwt-0009 fork.server.no-regression-normal-commands', () => {
  it('produces legacy-shape output for a minimal options object', () => {
    const out = mapCliOptionsToSDK({
      cwd: '/tmp/proj',
      permissionMode: 'default',
      toolsSettings: { allowedTools: ['Bash'], disallowedTools: [], skipPermissions: false },
    });

    expect(out.cwd).toBe('/tmp/proj');
    expect(out.allowedTools).toEqual(['Bash']);
    expect(out.disallowedTools).toEqual([]);
    expect(out.tools).toEqual({ type: 'preset', preset: 'claude_code' });
    expect(out.model).toBeDefined();           // defaults to CLAUDE_MODELS.DEFAULT
    expect(out.systemPrompt).toMatchObject({ type: 'preset', preset: 'claude_code' });
    expect(out.settingSources).toEqual(['project', 'user', 'local']);

    // No fork leakage:
    expect(Object.prototype.hasOwnProperty.call(out, 'resume')).toBe(false);
    expect(Object.prototype.hasOwnProperty.call(out, 'forkSession')).toBe(false);
    expect(Object.prototype.hasOwnProperty.call(out, 'resumeSessionAt')).toBe(false);
    expect(Object.prototype.hasOwnProperty.call(out, 'sessionId')).toBe(false);

    // 'default' permissionMode is intentionally NOT written through:
    expect(Object.prototype.hasOwnProperty.call(out, 'permissionMode')).toBe(false);
  });

  it('plan mode still unions in the plan toolset', () => {
    const out = mapCliOptionsToSDK({
      permissionMode: 'plan',
      toolsSettings: { allowedTools: [], disallowedTools: [], skipPermissions: false },
    });
    for (const t of ['Read','Task','exit_plan_mode','TodoRead','TodoWrite','WebFetch','WebSearch']) {
      expect(out.allowedTools).toContain(t);
    }
    expect(Object.prototype.hasOwnProperty.call(out, 'forkSession')).toBe(false);
  });

  it('skipPermissions → bypassPermissions (unless plan)', () => {
    const out = mapCliOptionsToSDK({
      toolsSettings: { allowedTools: [], disallowedTools: [], skipPermissions: true },
    });
    expect(out.permissionMode).toBe('bypassPermissions');
    expect(Object.prototype.hasOwnProperty.call(out, 'resume')).toBe(false);
  });
});
```

## Callers
- `mapCliOptionsToSDK`: single caller `queryClaudeSDK` (`server/claude-sdk.js:486`).
  Reuse risk: low.

## Anti-Patterns (DO NOT USE)
- Setting `sdkOptions.resume = undefined` explicitly — creates an own-property that fails `hasOwnProperty` checks.
- Using `{...input, ...sdkOptions}` shape which leaks arbitrary caller keys.
- Writing `sdkOptions.forkSession = Boolean(options.forkSession)` — forces the field to always exist.
